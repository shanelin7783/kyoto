# 京都旅遊行程 2026年5月
**成員：** 4大人 + 1小孩（小學三年級，約9歲）
**住宿：** [No.8 Kyoto](https://maps.google.com/?q=No.8+Kyoto+京都駅)（京都車站八條口步行**9分鐘**，約600m，5/23–5/26 共4晚）

- [x] **嵯峨野小火車**（5/25 13:05 トロッコ嵐山→トロッコ亀岡）✅ 已購票
- [x] **保津川遊船** 5/25 14:00 班次 ✅ 已預約
- [x] **Yamato Transport Namba Station Center（ヤマト運輸なんばステーションセンター）** 行李寄放（5/27 使用）✅ 已預約 09:00–17:30 ／ 難波4丁目4-4（距南海難波駅 步行2分鐘/140m）
- [ ] ⚠️ **購買近鐵特急指定席**（5/24 近鐵奈良→近鐵京都 18:34，大人4＋兒童1）— [近鉄 Web 購票](https://www.kintetsu.co.jp/railway/ticket/express_web/)
- [ ] 確認 5/25 晚餐餐廳（嵐山或京都市區）
- [ ] 確認 5/26 晚餐餐廳（四条河原町一帶）
- [ ] 備妥**現金**：清水寺不收刷卡；日本多處小店亦僅收現金
